# cnn-covid-xray-classification-cnn-covid-rayosx-clasificaci-n
Proyecto de aprendizaje profundo que implementa una red neuronal convolucional (CNN) para clasificar radiografías de tórax y detectar COVID-19 frente a pulmones sanos. Incluye preprocesamiento de imágenes, aumento de datos y balanceo de clases. Desarrollado para el curso Técnicas de Aprendizaje de Máquina.
